Write your content once, use it globally.

== Description ==

WP Global Shortcodes allows you to create content that can be globally used on as much places as you want with just maintaining it once.

= Features =

* Maintain Content once, use it globally
* Shortcode available
* Optimized permissions structure available
* Optimized development structure available
* Fully translateable
* Extendable structure

= For devs =

We offer a super extendable development structure for our WP Global Shortcodes plugin to optimize the code for your needs.

== Installation ==

1. Activate the plugin
2. Go to WP Global Shortcodes and start right away
3. You will find a new backend menu item called "Global Shortcode"

If you dan't see it, check your permissions
